Operators in Javascript



| Math operator | JavaScript |
| ------------- | ---------- |
| plus          | `+`        |
| minus         | `-`        |
| multiply      | `*`        |
| divide        | `/`        |
| power of      | `**`       |



| Assignment operators | JavaScript |
| -------------------- | ---------- |
| Equal                | `=`        |
| x = x + y            | x `+=` y   |
| x = x - y            | x `-=` y   |
| x = x * y            | x `*=` y   |
| x = x / y            | x `/=` y   |
| x = x + 1            | `++` x     |
| x = x - 1            | `--`x      |

 

| Comparison operators | JavaScript |
| -------------------- | ---------- |
| ﹥                   | >          |
| ﹤                   | <          |
| ≧                    | >=         |
| ≦                    | <=         |

> With comparison operators, you'll get a value holding boolean which means `true` or `false`.
